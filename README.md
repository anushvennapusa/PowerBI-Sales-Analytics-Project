# Sales Analytics Dashboard Project

**Tech Stack:** Power BI, SQL, Excel

**Project Objective:**  
Analyze sales data to visualize performance, profits, and regional trends.

---

## Folder Structure:

- **1_dataset/** → All Excel and CSV data files  
- **2_sql/** → SQL scripts and database  
- **3_powerbi/** → Power BI report file (.pbix)  
- **4_screenshots/** → Dashboard images

## How to Use:

1. Open Excel/CSV files in `1_dataset`  
2. Load the data into SQL using scripts in `2_sql`  
3. Open `Sales_Analytics_PowerBI.pbix` in Power BI  
4. View dashboards and analyze metrics
